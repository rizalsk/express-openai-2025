# Express Node.js App with TailwindCSS & OpenAI

## Overview
This is a Node.js application using **Express** framework with **TailwindCSS** for styling and integrated with **OpenAI API**. The project includes a development setup with live CSS watching and server restart.

---

## Requirements

- **Node.js:** 20.x
- **npm:** 10.8.2

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/rizalsk/express-openai-2025.git
cd express-openai-2025.git
```
Install dependencies:
```bash
npm install
```
Create a .env file in the root directory and add your environment variables (e.g., OpenAI API key):

```
OPENAI_API_KEY=your_openai_api_key_here
PORT=3000
```

# Development Server:
Run this command for development with live server and CSS compilation:
```bash
npm run dev
```
